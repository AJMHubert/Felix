Detailed Description

Module-Description: This is a generic fortran module for reading in data from dm3 files. It is generally not felix-specific but may specify types and formatting to match that are expected for felix code.

The following describes DM3 file format: https://imagej.nih.gov/ij/plugins/DM3Format.gj.html

Major-Authors: Jacob Richardson (2017)
